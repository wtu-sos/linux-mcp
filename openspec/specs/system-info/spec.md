## 新增需求

### 需求：获取 CPU 信息
系统必须从远程 Linux 主机获取 CPU 信息。

#### 场景：获取 CPU 信息
- **当** 请求 CPU 信息
- **则** 系统返回 CPU 型号、核心数和当前使用率

### 需求：获取内存信息
系统必须从远程 Linux 主机获取内存信息。

#### 场景：获取内存信息
- **当** 请求内存信息
- **则** 系统返回总内存、已用内存、可用内存和使用率

### 需求：获取磁盘信息
系统必须从远程 Linux 主机获取磁盘使用信息。

#### 场景：获取磁盘信息
- **当** 请求磁盘信息
- **则** 系统返回每个挂载文件系统的文件系统名、总大小、已用大小、可用大小和使用率

### 需求：获取操作系统信息
系统必须从远程 Linux 主机获取操作系统信息。

#### 场景：获取操作系统信息
- **当** 请求操作系统信息
- **则** 系统返回操作系统名称、版本、内核版本和架构

### 需求：获取系统运行时间
系统必须从远程 Linux 主机获取系统运行时间。

#### 场景：获取运行时间
- **当** 请求运行时间
- **则** 系统以人类可读格式返回运行时间（天、小时、分钟）

### 需求：获取全部系统信息
系统必须提供单个工具一次性获取所有系统信息。

#### 场景：获取全部系统信息
- **当** 请求系统信息摘要
- **则** 系统在单个响应中返回 CPU、内存、磁盘、操作系统和运行时间信息
